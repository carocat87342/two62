# two262-website

#### Run project

`./gradlew start`

This will start the project in dev environment.

#### Dependecies and External Libraries

Please use **yarn** to add/remove dependencies, **DO NOT** use **npm** unless you really have to.
 
`yarn add library`

`yarn remove library`
